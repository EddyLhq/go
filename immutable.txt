Go 和 Python 不可变数据类型的对比

特性	Go 中的不可变数据类型	      Python 中的不可变数据类型
字符串	  string	                str
整数	  无（Go 的整数是可变的）  	int
浮点数	  无（Go 的浮点数是可变的）	float
元组    无	                      tuple
布尔值	  无（Go 的布尔值是可变的）	bool
冻结集合	无	frozenset
数组	  array（长度和类型不可变）	无
常量	  const	                  无

Go 中的不可变数据类型
Go 的不可变数据类型较少，主要是 字符串 和 常量。
数组的长度和类型不可变，但元素值可以修改。

Python 中的不可变数据类型
Python 的不可变数据类型非常丰富，包括 字符串、整数、浮点数、元组、布尔值、冻结集合 等。
Python 的不可变性是其设计哲学的一部分，有助于简化代码和减少错误
